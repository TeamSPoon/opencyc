OpenCyc is the open source version of Cyc, the world's largest commonsense knowledge base.

These packages provide a high level deductive inference engine in the form of a finite domain constraint solver, together with KB object proxies and OpenCyc client interface support classes.

{@link org.opencyc.constraintsolver} provides the high level deductive inference engine to answer conjunctive queries.

{@link org.opencyc.cycobject} provides knowledge base proxy classes which OpenCyc applications must manipulate.

{@link org.opencyc.util} provides utility classes of general use.
